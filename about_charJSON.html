Instructions and test cases

 Here is a script that will take your csv file and return charJSON.

The original File must
  1.   have column names in the first row for EVERY value including the first column
  2.   have species or taxon names in the first column
It can have
  1.   numeric, strings, or boolean values
  2.   multiple character states, each separated by || i.e. two pipe characters
awk or gnuawk can be used.
  * for windows  you will need to install something like gawk
  * no software is required for macs (OS X) or linux

to use the script open a command window or terminal
get to the folder/directory in which you wish to work
type a command as follows
awk -F "," -f csv2json.awk YOUR_INPUTFILE >> YOURDESIRED_outputfile.json
    e.g.
awk -F "," -f csv2json.awk hornwort.csv >> hornwarts.json

awk -F "," -f csv2json.awk acacia.txt >> acacia.json
